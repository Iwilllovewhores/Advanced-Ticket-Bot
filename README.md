# 🎫 Advanced Ticket Bot

Um sistema de tickets moderno para Discord desenvolvido com **Discord.js v14**, focado em desempenho, organização e personalização.

> ⚠️ Este projeto ainda está em desenvolvimento e novas funcionalidades serão adicionadas constantemente.

---

## ✨ Recursos

* 📩 Sistema de abertura de tickets.
* 🛠️ Painel de configurações.
* 👥 Gerenciamento de cargos.
* 🔒 Fechamento de tickets.
* 📋 Sistema de permissões.
* ⚡ Desenvolvido com Discord.js v14.
* 🔧 Arquitetura modular.
* 🚀 Fácil personalização.

## 📥 Instalação

```bash
git clone https://github.com/Iwilllovewhores/Advanced-Ticket-Bot

cd Iwilllovewhores

npm install

npm run dev
```

---

## ⚙️ Configuração

Configure os arquivos necessários:

* Token do bot.
* IDs de cargos.
* Configurações do sistema de tickets.
